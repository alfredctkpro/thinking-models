## 想法與反思

### 114 年度所得稅申報研究

研究了夫妻報稅策略，結論：**太太薪資分開計稅**最有利。

#### 申報戶結構（8 人）

| 成員 | 年齡 | 免稅額 | 特別扣除 |
|------|------|--------|---------|
| 我 | — | 97,000 | 薪資 218,000 |
| 太太 | — | 97,000 | 薪資 218,000 |
| 父親 | 75+ | 145,500 | |
| 母親 | 74 | 145,500 | |
| 岳父（身障+長照） | 75+ | 145,500 | 身障 218,000 |
| 岳母 | 74 | 145,500 | |
| 小孩 1 | 6-12 | 97,000 | |
| 小孩 2 | 6-12 | 97,000 | |

- 免稅額合計：970,000
- 標準扣除額（夫妻）：262,000
- 薪資特別扣除額 × 2：436,000
- 身心障礙特別扣除額：218,000
- 基本生活費差額：254,000（213,000 × 8 人 vs 比較項合計）

#### 試算結果

| | 情境一（我 250 萬 + 太太 180 萬） | 情境二（我 300 萬 + 太太 180 萬） |
|--|----------------------------------|----------------------------------|
| 合併計稅 | 284,300 | 384,300 |
| **太太薪資分開（最優）** | **189,000** | **249,000** |
| 分開省下 | 95,300 | 135,300 |

#### 個人所得差異分析（太太薪資分開方案下）

| | 我的所得 | 我的課稅所得淨額 | 我的稅額 |
|--|---------|----------------|---------|
| 情境一 | 250 萬 | 675,000 | 39,700 |
| 情境二 | 300 萬 | 1,175,000 | 99,700 |
| **差額** | +50 萬 | +500,000 | **+60,000** |

多賺 50 萬，多繳 6 萬。這 50 萬全部落在 12% 稅率級距（59 萬～133 萬），邊際稅率 12%。

#### 關鍵發現

1. 兩個情境都是「太太薪資分開」最優，讓所得較低的太太分開（只帶走自己的免稅額 + 薪資扣除額），所有受扶養親屬的扣除額歸我合併計算。
2. 收入越高，分開計稅的節稅效果越明顯（95K vs 135K）。
3. 扶養人數多（8 人），基本生活費差額的保護效果很強，免稅額的增減會被 BLE 機制吸收。
4. 長照扣除額有排富限制（適用稅率 20% 以上不得使用），需注意。
5. 幼兒學前特別扣除額目前僅適用 6 歲以下，兩個小孩不符合。
6. 岳父雖為配偶之父，稅法上視同直系尊親屬，可報扶養並適用身障扣除額。
