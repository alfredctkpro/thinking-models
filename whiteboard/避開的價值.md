# 避開的價值

1. 後悔
   - 浪費時光的後悔
   - 作惡的後悔，對自己人格產生懷疑的後悔

2. 作惡 — 沒必要這樣折磨自己的身心
   - 莫貪心
   - 莫苟且
   - 莫偷雞

3. 大失敗
   - 缺乏勇氣的失敗
   - 遲遲不肯面對、拖延的失敗

4. 浪費時間 → 任何覺得浪費時間的事，就先 Let go 吧
