# 貝葉斯先驗——用證據更新你的成見

## 一句話摘要

概率是信念的度量，存在於人腦中；極大化所能找到的新證據，持續用它更新你的先驗（成見），才能趨近事情的真相。

## 模型說明

### 貝葉斯主義（Bayesianism）

貝葉斯主義提供了一套在不確定世界中做判斷的工具：

- **先驗（Prior）**：你預先的判斷，也可以說是你的成見、立場、先入為主
- **證據（Evidence）**：事件發生、資訊浮現，提供了新的觀察
- **後驗（Posterior）**：把證據帶入先驗之後，更新出來的新判斷

貝葉斯主義的核心主張：**概率是信念的度量，它存在於人的腦子裡，是對無知程度的一種量化**——你對這件事情到底瞭解多少。

因此，做判斷的關鍵不是「追求客觀事實」，而是極大化所能找到的新證據，用證據來改變先前的先入為主，進而更新你的後驗狀態。

### 兩個常見的錯誤

人類在運用這套工具時，常常犯兩個極端錯誤：

1. **完全沒有先驗**：宣稱自己沒有任何立場、純粹只看證據。但沒有先驗就沒有判斷的起點，事實上你仍會默認某種先驗，只是自己沒察覺。
2. **不願意動搖先驗**：即便已經看到了證據，仍不願更新。最極端的情況是把先驗概率直接設為 0 或 1（絕對不可能 / 絕對肯定），這在貝葉斯公式中等於永遠無法被任何證據更新——因為 0 乘任何數都是 0。

好的貝葉斯思考者，先驗堅定但永遠可動搖。

### 操作心法三步

1. **根據環境參數決定你的先驗**
   環境、脈絡、過往經驗，都是建立先驗的基礎。別假裝沒有成見——明確地寫下你現在的判斷。

2. **將發生的具體事情當作證據**
   不要用情緒或立場來解讀，把每一件發生的事情當成一筆資料。

3. **根據證據持續更新你的先驗**
   用證據修正判斷，形成新的後驗；下一次再遇到新證據時，這個後驗就是新的先驗。這是一個動態循環。

## 使用場景

- **面對新資訊**：判斷是否該改變既有看法，還是這只是噪音
- **評估他人論點**：檢視自己是否因為 0/1 信念而對證據視而不見
- **長期投資與決策**：在不確定環境中，用持續更新機制取代一次性判斷
- **與人爭論時**：意識到自己和對方都帶著先驗，爭的不是「事實」而是「先驗與證據的權重」
