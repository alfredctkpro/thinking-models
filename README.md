# 思維模型知識庫

個人知識管理系統，記錄思維模型、心智框架與每日反思。

主要來源：萬維鋼・得到 App

## 目錄結構

| 資料夾 | 用途 |
|--------|------|
| `models/` | 思維模型與心智框架 |
| `journal/` | 每日日記與反思 |
| `inbox/` | 快速捕捉，待整理筆記 |

## 模型分類

| 分類 | 說明 |
|------|------|
| [基本世界觀](models/基本世界觀/) | 對世界運作方式的根本認知 |
| [成長戰略](models/成長戰略/) | 個人成長與發展策略 |
| [決策判斷](models/決策判斷/) | 如何做出更好的決定 |
| [學習教育](models/學習教育/) | 學習方法與教育理念 |
| [財富增值](models/財富增值/) | 財富思維與增值方式 |
| [參與者](models/參與者/) | 如何有效參與和協作 |
| [領導者](models/領導者/) | 領導力與管理智慧 |
| [演化者](models/演化者/) | 演化思維與適應變化 |
| [高觀點](models/高觀點/) | 更高維度的思考視角 |

## 使用方式

### 記錄新的思維模型
1. 複製 `models/_template.md`
2. 放入對應分類資料夾，以模型名稱命名
3. 填入內容

### 寫每日日記
1. 在 `journal/YYYY/MM/` 下建立 `YYYY-MM-DD.md`
2. 參考 `journal/_template.md` 的格式

### 快速捕捉想法
1. 在 `inbox/` 下隨意建立 `.md` 檔
2. 之後整理到 `models/` 或 `journal/`
